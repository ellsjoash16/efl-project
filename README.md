# Elite Football League Manager

A football manager engine for a 12-club league, built for use alongside PES.

## Getting started

Open `index.html` directly in a browser, or run a local server:

```bash
npx serve . -p 3000
```

Then visit `http://localhost:3000`.

## Features
- League table with form, mood and prize money
- Free matchday fixture builder — you set who plays who
- Revenue & budgets tab with editable wage bills and transfer budgets
- Fan mood system driven by position, form and derby results
- Stadium manager with stand upgrades, seat expansion and rebuild options
- Sponsorship deals — add, edit, pause or remove
- Rivalry system with intensity tracking and H2H records
- End-of-season summary with next season rollover
